Multi-Angle Validation
This page is not for tuning one case manually — use the Single-Angle Tuning tab for that. It validates whether the PD gains currently set in the parameter panel (left) hold the pendulum upright across several initial angles at once, while also keeping the cart position x controlled. All other simulator settings (Ts, Tstop, limits, sensor quantization) are used as-is.
Each angle is simulated head-to-tail with identical gains; results appear below.
